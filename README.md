# Movie-ticket-booking-calculator
My first Python project: a movie ticket booking calculator using conditional logic

This is my first Python project, built while learning from freeCodeCamp.

Description:
This program simulates a movie ticket booking system. It calculates the final ticket prices based on different conditions such as age, membership status, show timin, and seat type.

Features: 
- Checks if the user is eligible to book a ticket
- Applies membership discounts
- Adds extra charges for weekends or evening shows
- Includes service charges for premium seats
- Calculates the final ticket price

  What I learned:
  - Using statements such as if and else
  - Working with logical operators
  - Writing clean ans structures Python code
